So essentially, I'm just doing this to give myself some kind of structure. I hope that by creating goals each day that I like and want to achieve, I will spend my days working towards those goals.

For now I've set this up as recurring tasks in linear, however I may like to somehow display my linear Todo and In Progress tasks over my desktop wallpaper as a widget/constant reminder.

## Desktop Wallpaper Overlay

This repo includes a small Conky setup that shows Linear tasks whose workflow state is `Todo` or `In Progress`. Tasks completed in the last 24 hours are also shown in green.
